# Art Direction

## Visual target

The target style is a clean, readable interior pixel look inspired by GBA and early DS-era RPG rooms: warm palettes, clear furniture silhouettes, strong tile readability, and cozy clutter without visual noise.

## Locked defaults

- Tile size: `16x16`
- Character sprite frame: `16x24` or `16x32`
- Camera: top-down
- Movement: four-direction only
- Scale: `4x` screen presentation

## Palette guidance

- Warm beige, muted stone, soft greens, and wood tones for interiors.
- Accent colors should be sparse and intentional.
- Floors and walls must contrast enough for collision readability.

## Object rules

- Objects must be legible at a glance.
- Collision should match the sprite silhouette closely enough to feel fair.
- Decorative density should never block route reading.
- Every room should keep a clear path between entry points and key interaction zones.
